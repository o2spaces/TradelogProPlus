ข้อมูลหายหลัง refresh browser ทำยังไง?▼
ข้อมูลถูกเก็บใน localStorage ของ Browser — ถ้าล้าง Cache / Incognito จะหาย วิธีแก้คือ (1) เชื่อม Google Sheet เพื่อ Sync อัตโนมัติ หรือ (2) กด 💾 Save to file เมนู ☰ สม่ำเสมอเพื่อ backup ไฟล์ .json
เชื่อม Google Sheet แล้ว Status ยังเป็นสีแดง?▼
ตรวจสอบ 3 จุด: (1) URL ถูกต้อง ขึ้นต้นด้วย https://script.google.com/macros/s/ (2) Deploy settings ตั้งค่า Execute as: Me + Anyone (3) ถ้าแก้ Code ใหม่ต้อง Deploy ใหม่เสมอ — ไม่ใช่ Save แค่ใน Editor
มีหลาย Portfolio ได้ไหม? ข้อมูลแยกกันไหม?▼
ได้ครับ — กด + พอร์ต สร้างได้ไม่จำกัด แต่ละ Portfolio มี Balance / Trades / Sessions แยกกันอิสระ ใน Google Sheet จะแยกด้วย Column "Portfolio"
AI Coach ใช้ข้อมูลของฉันไปเก็บไหม?▼
ไม่ — AI Coach ส่งข้อมูลตรงไปยัง Gemini / Groq API โดยใช้ Key ของคุณเอง TradeLog Pro+ ไม่มี Server กลาง ไม่เก็บข้อมูลใดๆ ทั้งสิ้น
Export ข้อมูลออกมาในรูปแบบอะไรได้บ้าง?▼
รองรับ 3 รูปแบบ: (1) JSON — เมนู ☰ → 💾 Save to file (2) CSV — เมนู ☰ → 📊 Export CSV (3) Google Sheet — เชื่อมต่อแล้วข้อมูลจะ Sync ขึ้น Sheet อัตโนมัติ
ใช้บนมือถือได้ไหม?▼
ได้ครับ — Responsive Design รองรับทุกหน้าจอ เปิดไฟล์ index.html บน Browser มือถือได้เลย แนะนำ Chrome หรือ Safari — บาง Feature เช่น File Import อาจมีข้อจำกัดบน iOS
Journal Mode กับ Data Mode ต่างกันยังไง?▼
Journal Mode: บันทึก trade จริงที่เทรดแล้ว ผูกกับ Portfolio, Balance, Equity Curve — เน้น Track Record
Data Mode: บันทึก Setup/Signal เพื่อวิเคราะห์เชิงสถิติ ไม่ผูกกับทุน — เน้น Strategy Research และหา Edge
Session Tag คืออะไร ตั้งค่าได้ไหม?▼
Session Tag คือป้ายกำกับ เช่น Sydney, Tokyo, London, New York — กำหนดได้ใน Sidebar ซ้าย (กดแท็บ Session Tags) + กด + Add เพิ่ม custom session ได้ ระบบจะ Auto-detect จากเวลา Entry ให้อัตโนมัติ

📂 วิธีที่ 2 — ผ่าน Google Drive

เปิด Apps Script และวางโค้ด
ใช้เวลาประมาณ 2-3 นาที
📂 เปิด Apps Script Editor
1ใน Google Sheet → คลิกเมนู Extensions ที่แถบเมนูบนสุด
2เลือก Apps Script → แท็บใหม่จะเปิดขึ้นมาพร้อม Code Editor
3จะเห็นโค้ดเริ่มต้น function myFunction() {} อยู่แล้ว
📋 วางโค้ดจากแอป
1กลับมาที่ TradeLog App → คลิก ⚙ → Google Sheet URL
2คลิกปุ่ม "📋 Copy Script" เพื่อคัดลอกโค้ด Apps Script ทั้งหมด
3กลับไป Apps Script Editor → กด Ctrl+A (เลือกทั้งหมด) → Ctrl+V วางทับ
4บันทึก โดยกด Ctrl+S หรือคลิกไอคอน 💾 ในแถบ Toolbar
▶️ รัน setupSheets() — ทำเพียงครั้งเดียว
1ที่แถบ Toolbar — คลิก Dropdown "Select function" → เลือก setupSheets
2คลิกปุ่ม ▶ Run → จะมี popup ขอสิทธิ์ → คลิก "Review permissions"
3เลือก Google Account → กด "Allow" → รอจนเห็น "Execution completed" ด้านล่าง
⚠️ หากเห็น "This app isn't verified"
คลิก "Advanced" (มุมล่างซ้าย) → "Go to [ชื่อโปรเจกต์] (unsafe)" → กด Allow
เป็นเรื่องปกติสำหรับ Script ส่วนตัวที่ยังไม่ได้ Publish ใน Google Marketplace
✅ หลัง setupSheets สำเร็จ — กลับไป Google Sheet จะเห็น Tab ใหม่:
Trades | DataTrades | Sessions | Summary | Assets | Journal | Log | DeletedPorts

🚀 ขั้นตอน Deploy ทีละขั้น
1ใน Apps Script Editor → คลิกปุ่ม "Deploy" มุมบนขวา

  • ตรวจ Quota: ใน Apps Script → View → Executions
